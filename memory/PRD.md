# Магазин грузовых автозапчастей - PRD

## Проблема
Создание интернет-магазина грузовых автозапчастей с каталогом, корзиной, оформлением заказов и личным кабинетом.

## Архитектура
- **Frontend**: React + Tailwind CSS + Shadcn UI
- **Backend**: FastAPI + MongoDB
- **Auth**: JWT tokens (bcrypt для паролей)

## Реализованный функционал

### v1.0 - MVP (16.01.2025)
- ✅ Главная страница с hero-секцией и 6 категориями
- ✅ Каталог товаров с фильтрацией по категориям
- ✅ Поиск по названию и артикулу
- ✅ Страница товара с добавлением в корзину
- ✅ Регистрация и авторизация пользователей
- ✅ Корзина (добавление, изменение количества, удаление)
- ✅ Оформление заказа (адрес, телефон, комментарий)
- ✅ Личный кабинет с историей заказов

### Категории товаров
1. Двигатель и комплектующие
2. Трансмиссия
3. Тормозная система
4. Электрика
5. Подвеска
6. Кузовные детали

## API Endpoints
- POST /api/auth/register - регистрация
- POST /api/auth/login - авторизация
- GET /api/auth/me - текущий пользователь
- GET /api/products - список товаров
- GET /api/products/:id - товар
- GET /api/categories - категории
- GET/POST/DELETE /api/cart - корзина
- GET/POST /api/orders - заказы
